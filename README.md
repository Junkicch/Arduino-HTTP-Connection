# Arduino-HTTP-Connection

  - Codigo de Arduino, onde ao receber um valor a placa manda um HTTP/POST com o valor de mensagem.
  - É necessario manter um Server aberto para ver a mensagem sendo enviada.
  - Placa usada Mega 2560.
  - módulo W5500.
  - Pinagem:
      - MI(MISO): 50
      - MO(MOSI): 51
      - CS: 10
      - SCK: 52
      - INT: 2
      - RST: RESET
